## Tasks

### Task 1: Verify order list page

Manual check (or screenshot):

- Verify DataTable renders with columns: Order ID, Customer, Retailer, Status, Total, Date
- Verify pagination works
- Verify status filter dropdown works
- Verify clicking an order navigates to order detail

### Task 2: Verify order detail page

- Navigate to /orders/:orderId
- Verify all cards render: Items, Payment, Delivery, Customer, Retailer, Address, Pricing
- Verify courier info shows in delivery card (if order has delivery data)
- Verify customer name links to user detail
- Verify retailer name links to retailer detail
- Verify order ID is copyable
- Verify back button works

### Task 3: Verify global search

- Click search button in header (or press Cmd+K)
- Type a user name → verify user results appear grouped
- Type a retailer name → verify retailer results appear grouped
- Type an order ID → verify order result appears
- Select a result → verify navigation to correct detail page
- Verify empty state when no results

### Task 4: Verify couriers page

- Navigate to /couriers
- Verify it shows informational content explaining courier data is within order details
- Verify link to orders page works

### Task 5: Build verification

- Run `npm run build` and confirm zero TypeScript errors
- Run `npm run lint` if configured
